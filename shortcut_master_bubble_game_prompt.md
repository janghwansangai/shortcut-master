# 🎮 "단축키 마스터" 버블 퍼즐 게임 개발 프롬프트

아래 내용 그대로 복사해서 바이브 코딩 툴(Claude Code, Cursor, v0 등)에 붙여넣어 사용하세요.

---

## 0. 한 줄 요약

기존 "색깔 맞추면 터지는" 버블 슈터(Bubble Shooter / Puzzle Bobble) 게임에서, **색 매칭 규칙을 "키보드 단축키 조합 매칭"으로 교체**한 캐주얼 학습 게임을 만들어줘. 플레이어는 버블을 터뜨리면서 자연스럽게 실제 키보드 단축키를 외우게 돼.

HTML5 Canvas + Vanilla JavaScript로 단일 HTML 파일에 구현해줘 (외부 빌드 도구 없이 브라우저에서 바로 실행 가능해야 함).

---

## 1. 기본 게임 틀 (버블 슈터 베이스)

- 화면 상단에 버블들이 **육각형 그리드** 형태로 매달려 있음 (벌집 구조, 한 줄씩 어긋남)
- 화면 하단 중앙에 **발사대(슈터)**가 있고, 마우스 움직임으로 조준 → 클릭/스페이스바로 발사
- 발사된 버블은 조준한 방향으로 직선 이동, 좌우 벽에 닿으면 반사(bounce)
- 다른 버블이나 천장에 닿으면 그리드의 가장 가까운 빈 칸에 자동으로 붙음(snap)
- 버블이 천장 기준선까지 쌓이면 **게임 오버**
- 그리드의 모든 버블을 제거하면 **스테이지 클리어**
- 일정 발사 횟수(예: 8회)마다 그리드 전체가 한 줄씩 아래로 내려와 압박감을 줘

---

## 2. ⭐ 핵심 규칙: 색 매칭 → 단축키 조합 매칭

### 2-1. 버블의 두 종류

| 종류 | 예시 | 비주얼 구분 |
|---|---|---|
| **보조키(Modifier) 버블** | Ctrl, Shift, Alt, Win | 진한 색 배경 + 사각형 느낌의 뱃지 모양, 키 이름 텍스트 |
| **일반키(Key) 버블** | C, V, X, Z, A, S, P, F, N, T, W, Esc, Delete, Tab 등 | 밝은 색 + 완전한 원형, 키 이름 텍스트 |

→ 두 종류는 **그리드와 슈터 양쪽에 무작위로 혼합되어 등장**해. 다음 발사할 버블은 화면 하단에 미리보기로 표시해줘 (기존 버블슈터의 "next bubble" UI와 동일).

### 2-2. 터지는 규칙 (색이 같으면 터지는 규칙을 대체)

1. 발사된 버블이 그리드에 붙는 순간, **인접한 6방향의 버블들**을 확인한다.
2. 새로 붙은 버블과 인접 버블의 키 조합이 **미리 정의된 "유효 단축키 테이블"**에 존재하는지 검사한다.
   - 예: 그리드에 `Ctrl` 버블이 있고, 내가 쏜 버블이 `C`인데 서로 인접하게 붙으면 → `Ctrl+C` 조합이 유효 테이블에 있으므로 **두 버블이 동시에 터짐**
3. 보조키 버블이 이미 2개 이상 체인으로 붙어 있는 경우(예: `Ctrl`+`Shift`가 인접), 거기에 맞는 일반키가 붙으면 **3중 조합**(`Ctrl+Shift+Esc` 등)도 터지도록 확장 가능하게 만들어줘.
4. 터진 버블 때문에 그리드 천장과 연결이 끊긴(공중에 뜬) 버블 덩어리는 자동으로 **낙하하며 사라짐** (콤보 점수 추가).
5. 매칭에 실패하면(인접 조합이 유효 테이블에 없으면) 버블은 그냥 그 자리에 붙어있는다 — 색 매칭 게임처럼 "틀려도 패널티 없이 다음 기회"로 넘어가는 캐주얼한 흐름을 유지해줘.

### 2-3. 유효 단축키 테이블 (예시 데이터 — 이대로 코드에 내장)

```
Ctrl + C        → 복사 (Copy)
Ctrl + V        → 붙여넣기 (Paste)
Ctrl + X        → 잘라내기 (Cut)
Ctrl + Z        → 실행 취소 (Undo)
Ctrl + S        → 저장 (Save)
Ctrl + A        → 전체 선택 (Select All)
Ctrl + P        → 인쇄 (Print)
Ctrl + F        → 찾기 (Find)
Ctrl + N        → 새로 만들기 (New)
Ctrl + W        → 탭/창 닫기 (Close Tab)
Ctrl + T        → 새 탭 열기 (New Tab)
Alt  + Tab      → 창 전환 (Switch Window)
Alt  + F4       → 창 닫기 (Close Window)
Win  + D        → 데스크톱 보기 (Show Desktop)
Win  + L        → 화면 잠금 (Lock Screen)
Win  + E        → 파일 탐색기 (File Explorer)
Ctrl + Shift + Esc → 작업 관리자 (Task Manager)
Ctrl + Shift + T   → 닫은 탭 복구 (Reopen Closed Tab)
```

→ 이 테이블은 배열/객체 형태로 코드 상단에 분리해서, 나중에 쉽게 추가/수정할 수 있게 만들어줘.

## 3. 🎉 터질 때의 "화려한 팝업" (이 게임의 핵심 재미 포인트)

두 버블이 합쳐져 터지는 즉시:

1. 터진 위치에서 **파티클/컨페티 효과**가 사방으로 퍼짐
2. 화면 중앙(또는 터진 좌표 근처)에 **뱃지형 팝업**이 등장
   - 큰 글씨로 키 조합 표시: 예) `Ctrl + C`
   - 그 아래 기능 설명: 예) `복사 (Copy)`
   - 어울리는 아이콘 (이모지로 대체 가능: 복사 📋, 저장 💾, 실행취소 ↩️ 등)
   - 등장 애니메이션: 스케일이 0 → 1.2 → 1.0으로 튕기듯(bounce) 커지며 등장
   - 1~1.5초 유지 후 위로 떠오르며 페이드아웃
3. 연속으로 여러 조합이 터지면(콤보) 화면 상단에 **콤보 카운터**(`COMBO x3!` 등)가 같이 표시되고 점수 가중치가 올라감
4. 가능하면 Web Audio API로 "터지는 소리"와 "팝업 등장 소리"를 간단한 합성음(beep/synth)으로 넣어줘 (외부 음원 파일 없이)

---

## 4. UI/UX 요구사항

- **상단 바**: 현재 점수, 콤보 수, 스테이지/레벨, 남은 발사 가능 횟수(선택)
- **하단**: 슈터, 다음 버블 미리보기, 조준선(점선 가이드)
- 일시정지 버튼 / 재시작 버튼
- 스테이지 클리어 시: "클리어! 오늘 배운 단축키" 모달로 이번 스테이지에서 터뜨린 조합들을 리스트로 다시 보여줌 (학습 효과 강화)
- 게임 오버 시: 최종 점수 + 다시 시작 버튼

---

## 5. 점수 및 난이도 곡선

- 단축키 조합 1개 성공 시 기본 점수 (예: 100점)
- 3중 조합 성공 시 더 높은 점수 (예: 300점)
- 콤보(연쇄 낙하)당 보너스 배율 적용
- 스테이지가 올라갈수록:
  - 그리드에 등장하는 단축키 종류 증가
  - 3중 조합 비중 증가
  - 그리드가 더 빠르게 내려옴

---

## 6. 기술 스택 / 구현 가이드

- 단일 `index.html` 파일에 `<canvas>` + `<script>` 포함 (외부 라이브러리 없이 Vanilla JS로 충분)
- 게임 루프는 `requestAnimationFrame` 사용
- 모바일 터치 지원: 터치 드래그로 조준, 탭으로 발사
- 반응형: 화면 크기에 맞춰 캔버스 비율 유지
- 코드는 다음 모듈로 구조화: `그리드/버블 데이터`, `충돌 판정`, `단축키 매칭 테이블`, `이펙트/팝업 렌더링`, `게임 상태 관리(점수/레벨/게임오버)`

---

## 7. 추가 아이디어 (선택 사항, 여유 있으면 구현)

- 카테고리별 테마 스테이지: "윈도우 기본", "크롬 브라우저", "포토샵", "VS Code", "엑셀" 등으로 다른 단축키 세트 적용
